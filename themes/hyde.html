Configuration interface

colorScheme

Description:

Selects the color scheme. Set to null for default black scheme.

Type:
null or one of "08", "09", "0a", "0b", "0c", "0d", "0e", "0f"

description

Description:

Content of the description meta tag.

Type:
string
Default:
"An elegant open source and mobile first theme for styx made by @mdo. Originally made for Jekyll.\n"

itemsPerPage

Description:

Number of posts per page.

Type:
integer
Default:
3

layout.reverse

Description:

Whether to enable reverse layout.

Type:
boolean
Default:
false
Example:
true
